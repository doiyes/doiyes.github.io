2020年观影记录

发表于 2020-03-28 分类于 other 阅读次数：
本文字数： 1.4k 阅读时长 ≈ 1 分钟
  • 釜山行 [★★★★★推荐] [2016韩国，类型: 动作 / 惊悚 / 灾难] 一句话介绍/点评：关于人性及一些细节闪光。
  • 现世活宝 [★★推荐] [1990中国大陆，类型: 剧情 / 喜剧] 一句话介绍/点评：赵本山早期作品，本山大叔很多小品段子的雏形都在里面。
  • 甲方乙方 [★★★★推荐] [1997中国大陆，类型: 喜剧] 一句话介绍/点评：好梦一日游，值得重温。
  • 一句顶一万句 [★★★推荐] [2016中国大陆，类型: 剧情 / 爱情 / 家庭] 一句话介绍/点评：经典台词：日子是过以后，不是过以前。。
  • 低俗小说 [★★★★★推荐] [1994美国，类型: 剧情 / 喜剧 / 犯罪] 一句话介绍/点评：神奇的1994年，经典的狐步舞。
  • 有完没完 [★★★推荐] [2017中国大陆，类型: 剧情 / 喜剧] 一句话介绍/点评：来看范伟的，至于其他人嘛？
  • 七宗罪 [★★★★★推荐] [1995美国，类型: 剧情 / 悬疑 / 惊悚 / 犯罪] 一句话介绍/点评：暴食、贪婪、懒惰、嫉妒、骄傲、色欲、愤怒。
  • 一声叹息 [★★★推荐] [2000中国大陆，类型:剧情 / 爱情 / 家庭] 一句话介绍/点评：典型性中国式婚外恋。
  • 只有芸知道 [★★推荐] [2019中国大陆，类型:剧情 / 爱情] 一句话介绍/点评：真人真事，但没看完。
  • 背对背，脸靠脸 [★★★★★推荐] [1994中国大陆，类型: 剧情] 一句话介绍/点评：官场现形记，一则中国社会的政治寓言，强烈推荐如果你能找到资源。
  • 美国往事 [★★★★★推荐] [1984美国，类型: 剧情 / 犯罪] 一句话介绍/点评：四个多小时，感觉还需要再看几遍。
  • 落叶归根 [★★★★★推荐] [2007中国大陆，类型: 剧情 / 喜剧] 一句话介绍/点评：本山大叔，世态炎凉，人情冷暖，看完又让人深思。
  • 过年 [★★★★推荐] [1991中国大陆，类型:剧情 / 家庭] 一句话介绍/点评：赵丽蓉、李保田、葛优等一众高手演绎最真实的中国式家庭。
  • 流感 [★★★★★推荐] [2013韩国，类型: 剧情 / 惊悚 / 灾难] 一句话介绍/点评：在2020这个特殊的春节里看着一部流感，天灾人祸，中国加油。
  • 放牛班的春天 [★★★★推荐] [2004法国，类型: 剧情 / 音乐] 一句话介绍/点评：小男孩的眼神、歌声，把我唱哭了。
  • 男妇女主任 [★★★★推荐] [2017中国大陆，类型: 剧情 / 喜剧] 一句话介绍/点评：赵本山，宋丹丹，笑果保证。
  • 蓝风筝 [★★★★推荐] [1993中国大陆，类型: 剧情] 一句话介绍/点评：文革题材，导致了导演的八年封禁。
  • 鬼子来了 [★★★★★推荐] [2000中国大陆，类型: 剧情 / 历史 / 战争] 一句话介绍/点评：大哥大嫂过年好！五舅姥爷、八婶子~。
  • 寄生虫 [★★★★★推荐] [2019韩国，类型: 剧情 / 喜剧] 一句话介绍/点评：2019年奥斯卡最佳影片。
  • 关连长 [★★★★推荐] [1951中国大陆，类型: 剧情 / 历史 / 战争] 一句话介绍/点评：非常真实，光头、不识字、山东话、为了幼儿园改变作战计划，这些“丑化”情节导致此片曾经被禁。
  • 一念天堂 [★★★推荐] [2017中国大陆，类型: 喜剧] 一句话介绍/点评：一个专门骗骗子的“网骗”惩恶扬善的故事。
  • 荒城记 [★★★★★推荐] [2018中国大陆，类型: 剧情] 一句话介绍/点评：李忆莲祠堂？礼义廉耻堂！蒋中正？讲忠贞！黑色幽默，荒诞不经！
  • 埋伏 [★★★★推荐] [2017中国大陆，类型: 剧情] 一句话介绍/点评：冯巩演的太真实了
